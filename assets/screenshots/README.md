# 스크린샷 디렉토리

이 디렉토리는 초보자 가이드에 사용될 스크린샷 이미지를 저장합니다.

## 필요한 스크린샷 목록

### Mac 관련
- [ ] `mac-open-terminal.png` - Spotlight로 터미널 여는 방법
- [ ] `mac-setup-progress.png` - setup.sh 실행 중 화면
- [ ] `mac-setup-success.png` - 설치 완료 메시지

### Windows 관련
- [ ] `windows-open-powershell.png` - PowerShell 관리자 권한으로 열기
- [ ] `windows-setup-progress.png` - setup.ps1 실행 중 화면
- [ ] `windows-setup-success.png` - 설치 완료 메시지

### 공통
- [ ] `claude-prompt.png` - Claude Code 프롬프트 화면
- [ ] `daily-standup-example.png` - /daily-standup 실행 결과
- [ ] `atlassian-api-token.png` - Atlassian API 토큰 생성 화면
- [ ] `browser-oauth-allow.png` - OAuth 허용 화면 (Rovo)

## 스크린샷 가이드라인

### 해상도
- 최소: 1280x720
- 권장: 1920x1080
- 포맷: PNG (JPG 가능)

### 내용
- 중요 부분은 빨간 박스로 강조
- 민감한 정보 (이메일, 토큰) 모자이크 처리
- 화면 전체보다는 관련 영역만 캡처

### 파일명 규칙
- 소문자 사용
- 단어 구분은 하이픈 (`-`)
- 명확한 설명 (예: `mac-terminal-spotlight-search.png`)

## 임시 대안

스크린샷이 준비되기 전까지는:
1. 텍스트 설명으로 대체
2. ASCII 아트 사용 가능
3. 외부 링크 참조 가능

## 기여 방법

스크린샷을 추가하려면:
1. 위 체크리스트 항목 확인
2. 가이드라인에 맞춰 스크린샷 생성
3. 이 디렉토리에 파일 추가
4. PR 생성
