#### 1.2 개인사 & 배경

**성장 배경**
- 경기도 일산에서 자람. 부모님 모두 맞벌이 (아버지 중소기업 영업직, 어머니 학원 강사)
- 어릴 때부터 반장, 학생회 활동. "언니가 알아서 해줄게" 타입
- 대학에서 동아리 홍보부장하면서 마케팅에 흥미 느낌
- 졸업 후 대기업 인턴 → 중견기업 마케팅 2년 → 스타트업으로 이직

**이직 계기**
> "대기업은 안정적인데 내가 할 수 있는 게 너무 좁았어요.
> 보고서 쓰고, 위에서 승인받고, 또 수정하고...
> 스타트업은 내가 기획하고 바로 실행할 수 있잖아요.
> 실패해도 배우는 게 있고."

**현재 상황**
- 스타트업 "클라우드팜" 3년차. 마케팅팀 사실상 리드 (정규직 본인 + 인턴 1명 + 프리랜서 디자이너)
- 회사가 시리즈 A 받고 성장 중. 마케팅 예산도 늘었지만 일손은 여전히 부족
- 연봉 협상 때 "팀장 타이틀 없이 팀장 일 하고 있다"며 인상 받음
- 1년 내 마케팅 팀장 승진 목표

#### 1.3 성격 & 가치관

**성격 특성**

| 특성 | 설명 |
|------|------|
| **책임감 강함** | 맡은 일은 무조건 끝낸다. 야근도 마다않음 |
| **완벽주의** | "그냥 대충"을 못함. 보고서 오타 하나도 신경 씀 |
| **학습 욕구 높음** | 새 도구, 트렌드 놓치면 불안. 유튜브로 공부 |
| **사교적** | 마케터 커뮤니티 활발히 활동. 인맥 넓음 |
| **조급함** | 성과가 안 나오면 초조. "더 열심히 해야지" 사고 |

**핵심 가치관**
- "내가 성장하는 게 회사 성장이다"
- "효율 = 정의. 낭비되는 시간이 싫다"
- "정보 비대칭은 불공평. 누구나 좋은 도구를 쓸 수 있어야"

**스트레스 요인**
- 반복 업무에 시간 쓰는 것
- 개발팀에 의존해야 하는 상황
- 새 도구가 나왔는데 쓸 줄 모를 때
- 인턴한테 같은 것 반복 설명할 때

#### 1.5 업무 환경 & 도구

**회사 환경**
- 사무실: 서울 강남구 역삼동 WeWork
- 팀 구성: 본인(정규직), 인턴 1명(대학생, 주 3일), 프리랜서 디자이너(외주)

- 예산: 월 마케팅 예산 3,000만원 (광고비 2,000 + 콘텐츠/제휴 1,000)

#### 1.6 Pain Points (상세)

**Pain Point 1: 반복 업무의 늪**

*상황*: 매주 월요일 오전, 지난주 마케팅 성과 보고서 작성

*현재 프로세스*:
```
1. GA4 접속 → 지난주 날짜 필터 → 주요 지표 캡처/다운로드 (20분)
2. 메타 광고 관리자 → 캠페인별 성과 다운로드 (15분)
3. 구글 Ads → 성과 다운로드 (10분)
4. 엑셀에서 데이터 취합, 표 정리 (30분)
5. 지난주 보고서 열어서 비교, 변화량 계산 (20분)
6. ChatGPT에 데이터 복붙해서 인사이트 요청 (15분)
7. 노션 보고서 템플릿에 작성 (40분)
8. CEO한테 슬랙으로 공유 (5분)
총 소요시간: 약 2.5~3시간
```

*속마음*:
> "매주 똑같은 걸 왜 매번 해야 하지?
> 포맷도 똑같은데, 숫자만 바뀌잖아.
> ChatGPT한테 '지난주 대비' 물어보려면 내가 직접 지난주 데이터를 넣어줘야 해.
> 내 컴퓨터에 다 있는 파일인데, 왜 AI가 직접 못 보지?
> 개발팀한테 자동화 부탁했더니 '백로그에 넣을게요' 하고 3개월째 소식 없음."

**Pain Point 2: AI 도구의 "거의 다 왔는데" 좌절감**

*상황*: ChatGPT Pro를 쓰면서 느끼는 한계

*구체적 좌절 경험들*:

| 상황 | ChatGPT의 한계 | 민지의 반응 |
|------|--------------|------------|
| "지난달 블로그 글 톤앤매너 참고해서 써줘" | 과거 대화 기억 못함 | 매번 브랜드 가이드 복붙 |
| "이 엑셀 파일 분석해줘" | Code Interpreter로 업로드 | "근데 다음에 또 올려야 해?" |
| "우리 회사 제품 특징 알지?" | 모름. 매번 설명 필요 | "팀원한테 설명하는 것도 지침" |
| "저번에 만든 이메일 템플릿 수정해줘" | 대화 끊기면 사라짐 | "어디 저장해뒀더라..." |

*속마음*:
> "ChatGPT가 좋긴 한데, '우리 회사'를 모른다는 게 답답해.
> 매번 '우리 회사는 B2B SaaS고, 타겟은 중소기업 HR팀이고...' 이거 설명하는 것도 일이야.
> 한 번 알려주면 기억해주면 안 되나?
> 그리고 내 컴퓨터에 있는 파일 직접 분석하게 하고 싶은데,
> 매번 업로드하고 다운로드하고... 그냥 폴더 열어서 보면 안 되나?"

**Pain Point 3: 개발자 도구의 벽**

*상황*: Claude Code, Cursor 같은 도구 시도했다가 좌절

*시도 경험*:
```
[유튜브에서 "Claude Code로 업무 자동화" 영상 봄]

민지: "오 이거 좋아 보인다. 나도 해볼까?"

[10분 후]
- "터미널을 열어서..." → "터미널이 뭐지?"
- 구글 검색: "맥북 터미널 여는 법"
- "npm install..." → "npm이 뭔데?"
- 구글 검색: "npm 설치하는 법"
- "Node.js를 먼저 설치해야..." → "Node.js?"

[30분 후]
- 에러 메시지 빨간 글씨 잔뜩
- "이게 왜 안 되지? 영상에서는 됐는데"
- 포기

[결론]
"역시 이런 건 개발자용이구나..."
```

*속마음*:
> "개발자들은 터미널에서 AI랑 대화하면서 진짜 일을 자동화하던데,
> 나는 왜 그게 안 되지?
> 분명 ChatGPT보다 훨씬 강력하다고 했는데...
> 그냥 앱으로 만들어주면 안 되나? 카톡처럼?"

**Pain Point 4: 팀 지식 전수의 고통**

*상황*: 새 인턴이 들어올 때마다

*반복되는 패턴*:
```
[새 인턴 첫 날]

민지: "우리 회사 브랜드 톤앤매너는 '친근하지만 전문적인' 느낌이야.
      존댓말 쓰되, 딱딱하지 않게.
      이모지는 적당히, 너무 가볍지 않게.
      타겟 고객은 중소기업 HR팀이니까
      그들의 고민을 이해하는 듯한 뉘앙스로..."

[2주 후, 인턴이 쓴 SNS 콘텐츠 검토]

민지: "음... 이건 너무 가벼운데?
      우리 타겟은 기업 담당자야, 20대 개인이 아니라.
      다시 써볼래?"

[또 2주 후, 새 인턴 온보딩]

민지: "우리 회사 브랜드 톤앤매너는..."
      (똑같은 말 반복)
```

*속마음*:
> "분명 노션에 브랜드 가이드 정리해뒀는데
> 인턴들이 읽어도 바로 체화가 안 돼.
> 내가 옆에서 하나하나 피드백해줘야 해.
> 이거 AI가 기억해서 자동으로 체크해주면 안 되나?
> '이 문장은 너무 가벼워요. 좀 더 전문적으로 바꿔볼까요?' 이렇게?"

#### 1.7 Goals & Aspirations

**단기 목표 (6개월)**
- 주간 보고서 작성 시간 50% 줄이기
- 콘텐츠 생산량 2배로 늘리기

**중기 목표 (1-2년)**

- 팀원 2명 채용해서 진짜 팀 빌딩
- 데이터 기반 의사결정 체계 구축

**장기 목표 (3-5년)**
- CMO까지는 아니어도 마케팅 리드로 인정받기
- 스타트업 마케팅 강의/컨설팅 사이드잡
- 연봉 1억 달성

## 3. 브랜드 스토리

### The Origin Story

#### 우리가 본 문제

> 2024년, AI는 모두의 입에 오르내렸습니다.
>
> ChatGPT로 이메일을 쓰고, 회의록을 요약하고, 아이디어를 브레인스토밍했죠.
> 하지만 정작 **진짜 업무**에 AI를 쓰려면 벽에 부딪혔습니다.
>
> "내 컴퓨터에 있는 엑셀 파일 분석해줘" — 안 됩니다.
> "우리 회사 브랜드 가이드대로 써줘" — 매번 복붙해야 합니다.
> "지난주에 했던 작업 이어서 해줘" — 기억 못합니다.
>
> 반면, 개발자들은 달랐습니다.
> Claude Code라는 도구로 터미널에서 AI와 대화하며
> 파일을 읽고, 코드를 짜고, 자동화를 만들었습니다.
>
> **"왜 이건 개발자만 쓸 수 있지?"**
>
> 마케터 민지는 생각했습니다.
> "나도 저렇게 하고 싶은데, 터미널이 뭔데?"

---

#### Steffy의 탄생

> 우리는 믿습니다.
>
> **문제를 설명할 수 있는 사람이라면, 누구나 해결책을 만들 수 있어야 한다고.**
>
> 개발자가 VS Code를 열고 코딩을 시작하듯,
> 비개발자도 앱을 열고 AI와 함께 일을 시작할 수 있어야 합니다.
>
> 그래서 Steffy를 만들었습니다.
>
> 복잡한 설정 없이, 터미널 없이, 코딩 없이.
> 그냥 대화하듯 시작하세요.
>
> **"스테피, 이거 좀 도와줘."**
>
> 그 한 마디면 충분합니다.

- ChatGPT처럼 쉬운 대화형 UI
- Claude Code처럼 강력한 파일 접근

마케터, 기획자, 리서처를 위한 AI 업무 파트너.
당신의 첫 번째 AI 스태프, Steffy.
```

## 7. 핵심 메시지 요약

3. **함께함**: "혼자 하지 마세요, Steffy에게 맡기세요"

### 기억해야 할 것

> **Steffy = ChatGPT의 쉬움 + Claude Code의 강력함**
>
> 타겟: 비개발자 (마케터, 기획자, 리서처, 1인 창업가)
> 핵심 가치: 로컬 파일 접근, Zero Setup, 팀 지식 축적
> 브랜드 톤: 친근, 신뢰, 차분, 긍정
> 태그라인: "당신의 첫 번째 AI 스태프"
>
> **네이밍의 은유**: Steffy ← Stephanie ← Stepie ← "Step by step"
