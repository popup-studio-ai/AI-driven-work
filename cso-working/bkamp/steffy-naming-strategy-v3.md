## 1. Steffy 네이밍 스토리

### 이름의 탄생 과정

```
Step by Step (단계별로 함께)
    ↓
Stepie (스테피) - step + ie
    ↓
Stephanie (스테파니) - 사람 이름 연상
    ↓
Steffy (스테피) - 친근한 애칭 형태
```

### 추가 연상 이미지

- **Staff** (스태프): 비서, 조수, 함께 일하는 동료
- **Stephanie**: 친근하고 신뢰감 있는 사람 이름

### 의미 레이어링

| 층위 | 연상 | 브랜드 메시지 |
|------|------|--------------|
| 1차 | Step by step | 단계별로 차근차근 도와주는 AI |
| 2차 | Stephanie | 친근한 사람 이름, 의인화 |
| 3차 | Staff | 유능한 비서/조수 이미지 |

## 2. Steffy vs Stepie 비교 분석

### 최종 비교

| 구분 | Steffy (스테피) | Stepie (스테피) |
|------|-----------------|-----------------|
| **발음** | 동일: 스테피 | 동일: 스테피 |
| **어원 연상** | Stephanie → 친근한 사람 이름 | step + ie → "단계" 직접 연상 |
| **철자 직관성** | 발음과 철자 일치 | "stepie"를 "스텝아이"로 읽을 수 있음 |
| **제품 의미 전달** | 간접적 (사람 → 도우미) | 직접적 (step by step) |
| **브랜드 느낌** | 세련됨, 친근함, 기억하기 쉬움 | 기능적, 명확함 |
| **검색 충돌** | Steffy 인명으로 일부 존재 | 상대적으로 적음 |

### 선택: Steffy

**이유**:
1. **철자-발음 일치**: "steffy"는 누구나 "스테피"로 읽음
2. **의미 레이어링**: step by step → stephanie → steffy 스토리가 풍부
3. **staff 연상**: 비서/스태프 이미지와도 연결
4. **글로벌 친화**: 영어권에서 친숙한 이름 형태
5. **브랜딩 확장성**: 캐릭터화, 의인화 마케팅에 유리

| **제품 연관성** | ⭐⭐⭐⭐⭐ (단계별 AI) | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |

### Steffy의 차별점

1. **제품과 이름의 연결**: "Step by step" = AI가 단계별로 도와주는 컨셉과 완벽 일치
2. **스토리텔링 풍부**: 이름의 유래를 설명하면 기억에 남음
3. **Staff 연상**: "유능한 스태프" 이미지로 B2B에서도 어필

## 4. 브랜딩 전략

### 핵심 메시지

```
"Steffy: Step by step, together"
- 단계별로 함께하는 AI 파트너
- 혼자 하기 어려운 일, Steffy와 함께
```

### 대안 태그라인

| 옵션 | 태그라인 | 강조점 |
|------|---------|--------|
| A | "Step by step, together" | 협업, 단계별 |
| B | "Your AI staff, Steffy" | 비서/조수 이미지 |
| C | "Work smarter with Steffy" | 생산성 |
| D | "Steffy: AI that walks with you" | 동반자 |

### 비주얼 아이덴티티 방향

**디자인 원칙** (토스 참고):

| 요소 | 방향 |
|------|------|
| **스타일** | 미니멀, 클린, 젠더 뉴트럴 |
| **컬러** | Primary Blue (#3182f6) 또는 네이비 |
| **타이포** | 모던 산세리프, 라운드 처리 |
| **아이콘** | 추상적 기하학 (S 또는 계단 모티프) |
| **느낌** | 전문적이면서 친근함 |

### 로고 컨셉 방향

1. **워드마크**: "steffy" 커스텀 레터링
2. **심볼**: 계단/스텝 모티프 또는 추상적 S
3. **조합형**: 심볼 + 워드마크
