## 브랜드 핵심 가치 제안

### 📌 One-Line Value Proposition

**"로컬 개발의 강력함과 자유로움, 클라우드 IDE의 편리함, 진짜 협업이 가능한 유일한 AI Agent"**

### 🎯 5대 핵심 가치

#### 1. **로컬 + Zero Setup** = 보안 + 자유 + 편의성
- 로컬 머신 실행 (코드 외부 유출 없음)
- 환경 설정 완전 자동화
- 오프라인 작업 가능

#### 2. **Claude Code의 강력함 + GUI** = 최고의 AI + 쉬운 UI
- Claude Code 네이티브 wrapper
- 200K 토큰 컨텍스트
- 터미널 두려움 제거

#### 3. **Atlassian 통합** = 진정한 팀 협업
- Jira & Confluence 네이티브 통합
- PM/디자이너/마케터도 사용 가능
- 티켓 → 코드 → 문서 자동화

#### 4. **플랫폼 독립** = 벤더락인 없음
- GitHub, GitLab, Bitbucket 모두 지원
- 어떤 호스팅도 자유롭게

#### 5. **프로토타입 → 프로덕션** = 확장 가능
- 컴포넌트 수 제한 없음
- 실제 서비스 배포 가능
