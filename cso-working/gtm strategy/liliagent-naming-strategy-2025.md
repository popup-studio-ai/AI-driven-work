## Executive Summary

### 핵심 상황
- **현재 이름**: Lili Agent (내부 사용)
- **목표**: 2025년 1월 공식 론칭을 위한 최종 네이밍 확정
- **제약**: .ai 도메인 확보 필수, 비개발자 친화적, 글로벌 확장 고려

### 제품 포지셔닝 (GTM 전략 반영)
```
"Claude Code 기반 AI Agent GUI Wrapper"
→ Zero Setup, 비개발자 친화, Atlassian 통합
→ bKend와 시너지로 End-to-End AI-Driven 개발 완성
```

### 타겟 사용자
- AI를 활용하고 싶은 빌더들
- 바이브코딩을 하는 크리에이터들
- PM, 디자이너, 마케터 등 비개발 직군
- 개발 환경 설정에 어려움을 겪는 사람들

## 네이밍 원칙 (2025 업데이트)

### 필수 원칙
1. **비개발자 친화적**: 기술 용어 지양, 쉬운 발음
2. **의인화 가능**: 사람 이름 느낌 (Siri, Alexa처럼)
3. **글로벌 확장성**: 한국어+영어 모두 자연스러운 발음
4. **.ai 도메인 확보**: [name].ai 또는 get[name].ai

### 부가 고려사항
5. **브랜드 메시지**: Zero Setup의 즐거움, 협업의 편리함
6. **bKend와 차별성**: Lili = 친근/협업, bKend = 전문/인프라
7. **기억 용이성**: 2음절, 명확한 스펠링

## Top 3 네이밍 추천

### 🥇 1순위: Lilli (getlilli.ai)

**도메인**: getlilli.ai ✅

**추천 이유**:
1. **브랜드 연속성**: Lili Agent → Lilli 자연스러운 진화
2. **기존 자산 활용**: 모든 내부 논의/문서 재사용 가능
3. **의미 완벽**: Lily(백합) = 순수, 아름다움, Zero Setup의 깨끗함
4. **팀 익숙도**: 이미 "Lili"로 불리고 있음
5. **Prefix 전략 검증**: Notion(notion.so), Linear(linear.app) 성공 사례

**브랜딩 메시지**:
```
"Lilli: Bloom your ideas"
- 백합처럼 순수한 개발 경험
- 아이디어를 꽃피우는 AI 파트너
- Get Lilli, Grow faster
```

**발음**:
- 한국어: "릴리" (매우 자연스러움)
- 영어: "Lil-lee" (친근하고 쉬움)

**도메인 전략**:
```
메인: getlilli.ai
대안: heylilli.ai, uselilli.ai
마케팅: "Get Lilli" (액션 유도)
```

**강점**:
- ✅ Zero Setup = 백합처럼 순수하게 바로 시작
- ✅ Claude Code의 강력함 + 아름다운 GUI
- ✅ 비개발자도 꽃을 피울 수 있다 (의미 메타포)
- ✅ 기존 논의 100% 재사용 (시간 절약)

**약점**:
- ⚠️ 도메인이 약간 김 (getlilli.ai)
- ⚠️ lilli.ai 직접 확보 불가 (혼동 가능성)

**종합 평가**: ⭐⭐⭐⭐⭐
- 브랜드 연속성 + 의미 완벽 + 팀 익숙도
- Prefix 도메인 전략은 이미 검증됨
- 즉시 실행 가능 (내부 합의 최소화)

---

### 🥈 2순위: Lettie (lettie.ai)

**도메인**: lettie.ai ✅

**추천 이유**:
1. **깔끔한 도메인**: lettie.ai 직접 확보 가능
2. **Lilli와 유사한 느낌**: 친근, 부드러움, 여성적
3. **의미 탁월**: Letitia(기쁨, 즐거움) = Zero Setup의 즐거움
4. **워드플레이**: "Let's..." / "Lettie..." 가능
5. **빈티지 클래식**: 우아하고 시대를 초월하는 느낌

**브랜딩 메시지**:
```
"Lettie: Let's create together"
- 함께 만드는 즐거움
- 개발의 기쁨을 경험하다
- Let Lettie handle the setup
```

**발음**:
- 한국어: "레티" (자연스러움)
- 영어: "Let-tee" (매우 쉬움)

**도메인 전략**:
```
메인: lettie.ai (깔끔!)
대안: getlettie.ai, heylettie.ai
```

**강점**:
- ✅ .ai 도메인 직접 확보 (간결함)
- ✅ "Let's..." 워드플레이 강력
- ✅ Lilli와 동일한 친근함/부드러움
- ✅ "기쁨" 의미 = 즐거운 개발 경험

**약점**:
- ⚠️ 새로운 이름 = 내부 적응 기간 필요
- ⚠️ 기존 "Lili" 자산 재사용 불가

**종합 평가**: ⭐⭐⭐⭐⭐
- 새로운 브랜드로 fresh start 원할 때 최적
- 깔끔한 도메인 + 의미 완벽
- "Let's create together" 메시지 강력

---

### 🥉 3순위: Ettie (ettie.ai)

**도메인**: ettie.ai ✅

**추천 이유**:
1. **초간결함**: 5글자, 2음절
2. **독특함**: 매우 드문 이름 (차별화)
3. **의미**: Henrietta(집의 주인) = 편안한 개발 환경
4. **Lilli 대체재**: 동일한 느낌 유지
5. **깔끔한 도메인**: ettie.ai 직접 확보

**브랜딩 메시지**:
```
"Ettie: At home with your development"
- 집처럼 편안한 개발
- 친숙하고 안전한 환경
- Your AI at home
```

**발음**:
- 한국어: "에티" (자연스러움)
- 영어: "Et-tee" (쉬움)

**도메인 전략**:
```
메인: ettie.ai
대안: getettie.ai
```

**강점**:
- ✅ 가장 짧고 간결 (5글자)
- ✅ 매우 독특해서 기억에 남음
- ✅ "집처럼 편안한" = Zero Setup의 편리함
- ✅ .ai 도메인 직접 확보

**약점**:
- ⚠️ 덜 알려진 이름 (설명 필요할 수 있음)
- ⚠️ "집의 주인" 의미가 약간 추상적

**종합 평가**: ⭐⭐⭐⭐
- 초간결 + 독특함 선호 시 최적
- Zero Setup = 집처럼 편안하게
- 차별화된 브랜드 구축 가능
