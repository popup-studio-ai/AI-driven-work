# bKamp는 무엇인가?

---

## 한 줄 정의

> **"AI 프로젝트의 마케팅 런치패드"**

바이브코더를 위한 커뮤니티 포털 + 쇼케이스 플랫폼

---

## 제품 개요

| 항목 | 내용 |
|------|------|
| **제품명** | bKamp (bkamp.ai) |
| **발음** | 비키 또는 비캠프 |
| **포지션** | AI 바이브코더 커뮤니티 포털 |
| **핵심 타겟** | AI로 뭔가 만든 사람들 (바이브코더) |
| **슬로건** | "AI로 만들고, 커뮤니티로 성장하라" |

---

## bKamp의 두 가지 얼굴

| 형태 | 설명 |
|------|------|
| **bkamp.ai (웹 포털)** | 쇼케이스 + 프롬프트 레시피 + 커뮤니티 |
| **bkamp agent (설치형)** | AI를 잘 쓸 수 있게 도와주는 로컬 Agent (구 릴리) |

> 하나의 브랜드, 두 가지 형태

# 우리가 해결하는 문제

---

## 기각된 가설들

### "정보가 파편화되어 AI를 시작 못한다"?

**❌ 틀렸다. 정보는 이미 넘쳐난다.**

| 플랫폼 | 규모 |
|--------|------|
| GPTers | **7,000개+** AI 활용사례 |
| AI 코리아 커뮤니티 | **12,311명** |
| YouTube "ChatGPT 사용법" | **수백 개** 영상 |
| 노마드코더 | **63만** 구독자 |

> 2009년 팟게이트가 성공한 이유는 "정보가 진짜 없어서"
> 2025년은 다르다. **정보 큐레이션만으로는 차별화 불가**

# Target Customer

---

## Primary: 바이브코더 (AI로 뭔가 만든 사람)

### 페르소나 1: 시리어스 빌더 "민수"
- 32세, 사이드 프로젝트 개발자
- Pain Point: "Product Hunt에 올렸는데 한국 유저 반응을 모르겠어"
- 현재 해결책: 디스콰이엇, 개인 SNS
- bKamp 사용 시: **한국어 피드백 + B2B 연결 + 투자 연계**

### 페르소나 2: 캐주얼 빌더 "지영"
- 28세, 마케터, Cursor로 첫 앱 제작
- Pain Point: "만들었는데 어디에 자랑하지? 피드백은 어디서 받지?"
- 현재 해결책: 없음 (개인 SNS 정도)
- bKamp 사용 시: **쇼케이스 등록 → 커뮤니티 피드백 → Featured 선정**

---

## Secondary: 바이브코딩 입문자

### 페르소나 3: 입문자 "서연"
- 24세, 대학생
- Pain Point: "바이브코딩 해보고 싶은데 어디서 시작하지?"
- 현재 해결책: YouTube, ChatGPT
- bKamp 사용 시: **입문 가이드 → 튜토리얼 → 첫 프로젝트 → 쇼케이스 등록**

---

## Seed Users: 제로백 동문

- 이미 팝업스튜디오와 관계 있음
- 즉시 유입 가능
- 콘텐츠 시딩 역할

## 4대 핵심 기능

| 기능 | 설명 | 핵심 가치 |
|------|------|----------|
| **쇼케이스** | AI 앱 등록/조회/투표 (베스트도전) | 노출, Upvote를 통한 인정 |
| **프롬프트 레시피** | AI 도구 조합 워크플로우 공유 | 실전 노하우 공유 |
| **커뮤니티 Q&A** | 프로젝트별 질문/답변, 피드백 | 실질적 피드백, 지식 공유 |
| **마케팅 부스팅** | Featured 선정 → SNS/뉴스레터 홍보 | 노출 + B2B 연결 |

## 쇼케이스 승격 시스템 (베스트도전)

```
[1단계: 등록]
    │
    ├── GitHub 링크 제출
    ├── 스크린샷/데모 영상 업로드
    └── 어떤 AI로 만들었는지 설명
          │
          ▼
[2단계: 커뮤니티 검증]
    │
    ├── 공감(Upvote) 받기
    └── 댓글/피드백
          │
          ▼
[3단계: 호스팅 제공] ← 공감 50+ 달성 시
    │
    └── 포털 내 정적 호스팅
          │
          ▼
[4단계: 하이라이트 승격] ← 공감 200+ 달성 시
    │
    ├── 메인 페이지 노출
    ├── bKend 백엔드 연동 지원
    └── 마케팅 지원
          │
          ▼
[5단계: 정식 런칭]
    │
    └── 수익화 연동
```

# bKamp + bKend 시너지

---

## 통합 가치

```
bKamp (커뮤니티 포털)          bKend (백엔드)
      ↓                              ↓
프로젝트 등록             ←→      백엔드 제공
커뮤니티 검증             ←→      API 연동
Featured 승격             ←→      인프라 지원
      ↓                              ↓
           마케팅 + 사업화 지원
```

### 승격 시나리오
1. 빌더가 bKamp에 프론트엔드 프로젝트 등록
2. 커뮤니티 검증 (공감 200+)
3. **하이라이트 승격 → bKend 백엔드 연동 지원**
4. 정식 서비스 런칭
