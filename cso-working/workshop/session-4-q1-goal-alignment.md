### MVP (2주, 런칭 시점)

|------|------|

| 사용자 시스템 | Google OAuth 로그인 |
| 커뮤니티 Q&A | 프로젝트별 질문/답변 |

### Phase 2 (1월)
| 기능 | 설명 |
|------|------|

| Featured 시스템 | 주간 우수작 선정 |
| GitHub OAuth | 추가 인증 옵션 |
| 검색 기능 | 프로젝트/레시피 검색 |

### Phase 3 (2-3월)
| 기능 | 설명 |
|------|------|
| bKend 연동 | 하이라이트 승격 시 백엔드 지원 |
| 호스팅 제공 | 공감 50+ 프로젝트 정적 호스팅 |
| 마케팅 부스팅 | SNS/뉴스레터 자동화 |
| Admin 대시보드 | 운영 관리 도구 |
