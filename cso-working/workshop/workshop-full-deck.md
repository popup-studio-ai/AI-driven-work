# 우리가 해결하는 문제

---

## Vibe Coding의 현실

### 문제: 백엔드의 벽
- 프론트는 10분, 백엔드는 10일
- 인증, DB, API... 매번 반복
- AI가 만들어줘도 배포가 문제

---

## 현재 대안들의 한계

| 대안 | 장점 | 한계 |
|------|------|------|
| Supabase / Firebase | 빠른 시작 | Multi-tenancy 직접 구현 |
| Vercel / Railway | 쉬운 배포 | 백엔드 로직은 직접 구현 |
| 직접 개발 | 완전한 컨트롤 | 시간 많이 소요 |

> **bKend = Vibe Coding의 마지막 퍼즐**

# bKamp + bKend 시너지

---

## 통합 가치

```
bKamp (커뮤니티 포털)          bKend (백엔드)
      ↓                              ↓
프로젝트 등록             ←→      백엔드 제공
커뮤니티 검증             ←→      API 연동
```
